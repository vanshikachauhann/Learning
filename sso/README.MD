# SSO(Single sign on)
It is an authentication method where a user logs in once and gets access to multiple applications or systems without logging in again and again.

#   How SSO Works (Simple Flow)

     * User tries to open an application
     * User logs in once
     * Server verifies identity
     * Server sends a trust token back to apps
     * User can now access all connected apps

# Advantages of SSO

            * Only one login needed
            * Saves time
            * Better user experience
            * Fewer passwords to remember
            * Centralized security control
            * Easy for companies to manage users
